# HikCentral Biometric Attendance Dashboard 

[![Framework - Next.js](https://img.shields.io/badge/Framework-Next.js%2015-black?style=for-the-badge&logo=nextdotjs)](https://nextjs.org/)

[![Styling - Tailwind CSS](https://img.shields.io/badge/Styling-Tailwind%20CSS-06B6D4?style=for-the-badge&logo=tailwindcss)](https://tailwindcss.com/)
[![Language - TypeScript](https://img.shields.io/badge/Language-TypeScript-blue?style=for-the-badge&logo=typescript)](https://www.typescriptlang.org/)

A real-time logging portal and analytics console built with **Next.js 15 (App Router)** and **Supabase**, engineered to act as a cloud-native viewing console for an on-premise **HikCentral Access Control** deployment.

This application replaces legacy Excel-based attendance exports with a live-updating, interactive control panel optimized for privileged accounts, HR administrators, and managers for payroll management.
